spring tool suite4的总结

  • 2019-05-16
  • 权静

eclipse：

IBM贡献的，Java语言开发的，最初主要是Java集成开发环境（IDE），开源免费，插件灵活，Java开发工具（java development kit, JDK插件 ），其他计算机编程语言也能很好的支持

spring tool suite：

基于eclipse针对spring的开发环境，git maven

####区别
eclipse、spring界面不同，操作一样

####使用

  1. 右键import folder from existing projects，选择文件，配置好属性配置文件，一般只需要我们改一下数据库名密码为自己的
    application.properties（系统默认）或者application.yml文件是属性配置文件，可以设置多环境配置，写多个application-dev.properties/application-pro.properties/application-test.properties,需要操作哪个环境，直接命令行启动时，修改主文件application.properties的spring.profiles.active=dev就可以：看云 http://blog.didispace.com/springbootproperties/
  2. 找到带有main方法的类的Java文件，main方法里会有一个springApplication.run函数，运行这个类:
    运行spring boot 的方式：
  3. 运行带有main方法的类
  4. 安装maven后，项目打包成jar，运行命令
    cd 到项目根目录
    mvn install安装maven
    cd target
    java -jar emample.jar –server.port=8081
    如果pom文件有spring-boot-maven-plugin语句，可以使用命令行运行，mvn springboot:run
    #####–
    代表设置参数，会覆盖掉application.properties里的设置，可以以此来改变不同端口，而不同端口是不同的运行环境：Dev，Pro，test
    #####项目根目录下：
    pom.xml文件：配置项目依赖，类似package.json文件
    #####comd+shift+g:
    全局查找：~/.m2下，settings.xml文件，是maven的配置文件
    maven build, maven install, maven clean
    #####pom.xml文件：
  5. groupId:
    整个大项目的名称，com.quanjing.haha:域 + 公司名 + 整个项目名
  6. artifactId:
    具体项目的名称： haha-tolink
  7. version: 版本
  8. packaging：
    规定项目输出格式，一般Java项目：jar，Android项目apk
    java version “1.8.0_201”和jdk8：

java version “1.8.0_201”是完整的版本字符串

jdk8或者Java™ SE Development Kit 8, Update 201 (也简写JDK 8u201)是family version，家族昵称

都是指同一个Java版本

现在普遍是jdk8
